# THE MAIN — Telegram-бот подписок + админка

Бот продаёт подписки на закрытые ресурсы (приватный канал/форум/чат) с тарифами
1 / 3 / 6 / 12 месяцев и «Навсегда», ведёт срок по московскому времени, поддерживает
автопродление и его остановку, подарочные подписки и доступ по заявкам с одобрением ботом.
